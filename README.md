 aulas online (ou outra coisa qualquer) num unico local

1 - Ir a "code" (a verde) e **"download ZIP"**

(Alternativamente, ir às versões, no lado, e escolher a versão desejada)

2 - Abrir o NMAIAS.pyw

Python 3.7+
